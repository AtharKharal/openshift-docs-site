OpenShift Architecture

This page outlines the architectural building blocks of OpenShift, including control plane and worker nodes.

1. Core Components

Component Function

API Server

Handles REST requests and exposes the Kubernetes API.

Controller Manager

Runs background tasks and manages desired state.

etcd

Key-value store used for persistent cluster configuration.

Scheduler

Assigns workloads (pods) to nodes.
